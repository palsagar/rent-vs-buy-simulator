# Results page is a verdict narrative, not a dashboard

The results side previously spread the answer across five metric tiles and five peer tabs. It is now a single scrolling narrative: verdict hero (Verdict + Breakeven + Confidence in plain language, numbers untruncated) → the Net Value decision chart with breakeven marker → "How sure is this?" (fan + tornado) → "Where the money goes" (outflows + ownership-cost breakdown) → collapsed data table.

The Asset Growth chart is deliberately removed, not lost: a gross home-value line ignores debt, outflows, and exit costs, and visually contradicts the Verdict under liquidation-based Net Value (ADR-0001). Do not reintroduce gross asset-value visuals.
